Quick lesson algorithm:

1. Show question
2. Await answer
3. Mark answer
4. If correct   => Show green 
                => Allow user to proceed to next question
4. If incorrect => Show red
                => Require user to type correct answer
                => Allow user to proceed to next question

5. If no more questions remain => Show statistics
6. If more questions remain    => Go to next question

Memorisation lesson algorithm: Uses the quick lesson algorithm in 3 stages.

1. Multiple choice [georgian -> english]
2. Intermission
3. Translation     [georgian -> english]
4. Intermission
5. Translation     [english  -> georgian]
6. Show post lesson statistics.
